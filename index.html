Cloud sync
Save data to a private GitHub Gist so it survives across devices. How do I set this up?
  1. Go to github.com/settings/tokens?type=beta, generate a fine-grained token with Gists: Read and Write.
  2. Go to gist.github.com, create a secret gist with file track.json containing {}. Copy the long ID from the URL.
  3. Paste both below.
Auto-log Zone 2
Workouts pushed from your Apple Watch are logged as Zone 2 sessions when the average heart rate lands in the window below. How do I set up the Shortcut?
  1. On iPhone, open Shortcuts → Automation → +, trigger: When any Apple Watch workout ends, Run Immediately.
  2. Actions: Find Health Samples (Workouts, latest 1) → Find Health Samples (Heart Rate, between the workout's start and end) → Calculate Statistics: Average.
  3. Text action with: {"start":"[Workout Start]","duration_min":[Workout Duration in minutes],"avg_hr":[Average],"type":"[Workout Type]"}
  4. Get Contents of URL: PATCH https://api.github.com/gists/YOUR_GIST_ID, header Authorization: Bearer YOUR_TOKEN, JSON body: files → watch-inbox.json → content = the Text.
  5. Same gist and token as Cloud sync. The app ingests it within ~30s while open, or on next launch. Full recipe: apple-watch-setup.md in the repo.
